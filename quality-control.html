Контроль качества

Система менеджмента качества

Работа с металлами и газами, сложность технологического процесса требуют от предприятий нефтегазовой отрасли обеспечения высокого уровня безопасности персонала и окружающей среды, необходимой эксплуатационной целостности, а также высочайшего качества выполнения работ и предоставления услуг.

В компании Шоллер Блэкманн Даррон все это гарантируется системой менеджмента качества, которая с успехом применяется на предприятии с 2011 года. Система менеджмента качества Шоллер Блэкманн Даррон направлена непрерывное усовершенствование, предотвращение дефектов и сокращение количества несоответствий стандартам качества. Сертификация компании по стандартам ISO 9001:2008 и API Specification Q1 Edition 9 позволяет гарантировать высокое качество изделий и услуг, предлагаемых компанией. Лицензия API Monogramm дает компании право маркировать монограммой API свою продукцию, изготовленную в соответствии с требованиями API Specification 7-1

Политика в области качества компании

Компания «Шоллер Блэкманн Даррон» в России ориентирована на предоставление услуг и поставку продукции высочайшего качества в целях удовлетворения потребностей Заказчиков и постоянного повышения уровня качества работы предприятия. Чтобы достичь своих целей, Компания должна соответствовать всем требованиям системы менеджмента качества и постоянно работать над ее эффективностью, весь персонал должен быть ознакомлен с требованиями стандартов качества, а также со всеми инструкциями, формами и процедурами, имеющими отношение к должностным обязанностям. В связи с ужесточением требований заказчиков и высокой конкуренцией на рынке производства и продажи оборудования для предприятий нефтегазовой отрасли, персонал Компании прикладывает максимум усилий для повышения качества продукции, услуг, технологий, систем и методик усовершенствования результатов работы. Специалистами компании была разработана и внедрена Система Менеджмента Качества, соответствующая требованиям стандарта ISO 9001: 2008, API Q1 Spec 7 – 1.
